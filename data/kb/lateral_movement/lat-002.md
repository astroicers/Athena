Pass the Ticket attacks leverage stolen Kerberos Ticket Granting Tickets (TGTs) or Service Tickets to authenticate to resources in an Active Directory environment without knowing the account's password. TGTs are extracted from LSASS memory or obtained through delegation abuse using tools like Mimikatz or Rubeus, then injected into the current logon session to impersonate the legitimate user across the network. Golden Ticket attacks extend this technique by forging TGTs signed with the KRBTGT account hash, granting indefinite, domain-wide access even after password resets on the target accounts.
