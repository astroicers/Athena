Pass the Hash (PtH) is a lateral movement technique that authenticates to remote systems using a captured NTLM password hash without cracking it, exploiting the fact that NTLM authentication transmits a hash-based challenge response rather than the plaintext password. Hashes are typically extracted from LSASS memory on a compromised host using Mimikatz's `sekurlsa::logonpasswords` command (requires SYSTEM or debug privileges), then used to authenticate to other machines in the network where the same account exists. The technique works because Windows NTLM authentication only validates that the client possesses the correct hash, making credential harvesting and reuse trivial across environments without password expiration enforcement on local admin accounts.
