Evil-WinRM is a tool that provides a feature-rich interactive PowerShell shell over the Windows Remote Management (WinRM) protocol on ports 5985 (HTTP) and 5986 (HTTPS), which is enabled by default on Windows Server 2012 and later. It supports authentication with passwords, NTLM hashes (pass-the-hash), and SSL certificates, and includes built-in functionality for file upload/download, in-memory PowerShell script loading, and bypassing AMSI. WinRM access requires that the connecting user is in the local Administrators group or the Remote Management Users group, making it a useful lateral movement path when such credentials have been compromised.
